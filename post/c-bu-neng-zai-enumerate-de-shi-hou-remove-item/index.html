C# 不能在enumerate 的时候remove item

2020-11-17

今天遇到个问题，需要在ObservableCollection里面根据一个item的Name移除它，想着就写了然后就报了错，大致意思就是不能在enumerate 的时候remove item，因为remove item后Collection就改变了就不能enumerate了。之后找到的解决办法是，加一个break关键字就好，因为反正我也只需要移除一个item，所以刚刚够用。

 foreach (var detector in Detectors)
 {
         if (detector.Name==selectedItemName)
           {
               Detectors.Remove(detector);
                break;
            }
 }

还看到Stackoverflow上说，可能和multiple-thread有关，我觉得是有道理的，但也还没有特别明白，先这样吧！
